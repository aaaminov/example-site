Чеклист публикации на GitHub Pages

  1. Создайте публичный репозиторий на GitHub.
  2. Загрузите все файлы из этой папки.
  3. Откройте Settings, затем Pages.
  4. Выберите источник: Deploy from a branch.
  5. Укажите ветку: main, папку: /root.
  6. Дождитесь появления URL сайта.

Не публикуйте личные данные. Размещайте только безопасный учебный контент.
