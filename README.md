# Web Sprint Lab

Small multi-page website example for school lessons.

## Included pages

- index.html
- topics.html
- projects.html
- contact.html

## Files

- styles.css (shared styles)
- .nojekyll (optional marker for GitHub Pages)

## Run locally

Open index.html in a browser.

## Publish to GitHub Pages

1. Create a public repository on GitHub.
2. Upload all files from this folder to the repository root.
3. Open repository Settings -> Pages.
4. In Build and deployment, choose Source: Deploy from a branch.
5. Select Branch: main and Folder: /root.
6. Save and wait 1-2 minutes.
7. Open the generated site URL from the Pages section.

## Notes for students

- Keep all page links relative (for example: topics.html).
- Do not publish private personal data.
- Use one shared styles.css file for all pages.
