Lesson flow

Topics to cover in one practical class

This page can be used as a board plan: students see the exact structure of the work and move step by step.

Core HTML blocks

  • Document structure: html, head, body
  • Text: h1-h3, p, strong, em
  • Lists: ul, ol, li
  • Links: internal navigation between pages
  • Media: image with alt text

Practice checkpoints

  1. Create 4 HTML files in one folder.
  2. Add the same menu on every page.
  3. Check all links by clicking each item.
  4. Apply one shared CSS file for all pages.
  5. Push to GitHub and publish via Pages.

Suggested pacing (90-100 minutes)

10 min Safety and recap of previous lesson

15 min Teacher live coding on projector

40 min Student individual work

20 min Pair checks and fixes

10 min Demo of GitHub Pages deployment
