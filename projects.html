Идеи для учеников

Быстрые темы для личного многостраничного сайта

Каждую тему ниже легко завершить за одно занятие, а позже расширить с помощью CSS и JavaScript.

Клуб моего хобби

Расскажите о своем хобби, инвентаре, топ-5 советах и полезных ссылках.

Подходит для: спорт, музыка, рисование

Гайд по игре

Сделайте мини-гайд с обзором игры, списком персонажей и стратегией для новичка.

Подходит для: геймеров и зрителей стримов

Гид по городу

Создайте страницу о своем городе с интересными местами и локациями для фото.

Подходит для: проектов о местной культуре

Будущая профессия

Опишите профессию мечты, необходимые навыки и личный план обучения.

Подходит для: профориентации

Рекомендуемая структура сайта

  1. Главная: название, короткое описание, меню навигации.
  2. Обо мне: детали, список интересов, одно изображение.
  3. Проекты: 2-3 карточки с описаниями.
  4. Контакты: безопасные публичные ссылки и чеклист публикации.

Выберите идею и опубликуйте свою версию сайта.
